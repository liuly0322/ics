lc3tool #

LC3 模拟器

在 Github Release 内可以下载到最新版本的软件

  • 如果网络环境不支持直接下载，可以尝试使用如下的 ghproxy 链接：
    • MacOS：点此下载
    • Linux(AppImage)：点此下载
    • Windows：点此下载

使用说明：http://acsa.ustc.edu.cn/ics/download/lc3/GuideToUsingLC3Tools.pdf

点击 View 后选择 Zoom in 和 Zoom out 来缩放 UI 大小

对应快捷键分别是 Ctrl Shift + 和 Ctrl -
